# Damascan Daydreams Website

A clean, responsive website for the musical project Damascan Daydreams.

## Features

- Minimalist dark theme design
- Mobile-responsive layout
- Sections for music, videos, shows, biography, photos, and contact
- Easy to maintain and update

## Pages

- **Home/Music**: Showcases music releases with streaming links
- **Videos**: YouTube embeds of music videos
- **Shows**: Listing of upcoming performances
- **Bio**: Artist biography
- **Photos**: Gallery of images
- **Contact**: Contact information

## Development

This is a static HTML/CSS website using:
- HTML5
- CSS3
- Minimal JavaScript for dynamic content

## Deployment

This site is hosted via GitHub Pages. To deploy changes:

1. Clone this repository
2. Make your changes
3. Commit and push to the main branch
4. GitHub Actions will automatically deploy the changes

## Local Development

To run the site locally, simply open any HTML file in a web browser or use a local server:

```bash
# If you have Python installed
python -m http.server

# If you have Node.js installed
npx serve
```
